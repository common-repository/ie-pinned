This plugin allows you to make your blog pinnable in IE9.

== Description ==

IE Pinned adds the basic pinnable feature to your blog. What this does is allow users of IE9, when your site is pinned to the taskbar, be able to view your latest posts and pin them to be there all the time.

It will also allow you to have your categories and menu on there.

== Installation ==


1. Upload `ie-pinned` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Settings' menu and select 'IE Pinned Settings' option to customise.

== Frequently Asked Questions ==

= Where do I put my favicon =

Inside the IE Pinned plugin folder. Future update will be to do this by the admin section.

= My tooltip and other parts aren't working =

This is a problem with IE9 itself, mainly the 64 bit edition of Windows 7 IE9 has the most problems especially the 64 bit edition of IE9. I have contacted Microsoft about this to help fix these bugs.
